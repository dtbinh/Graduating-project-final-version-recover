Orientação para instalação do Latex no Windows:
-------------------------------------------------------

1 - Download e instalação do software Ghostscript (http://www.ghostscript.com/download/gsdnld.html)

2 - Download e instalação do visualizador Ghostview ou GSview (http://pages.cs.wisc.edu/~ghost/)

3 - Download e instalação de visualizador pdf caso não o tenha

4 - Download e instalação do software MikTeX (http://miktex.org/download): versão Basic (os pacotes serão baixados sob demanda, necessário conexão internet para compilar) ou versão completa (download all packages and install a complete TeX/LaTeX system)

5 - Download e instalação do editor Texstudio (http://texstudio.sourceforge.net/)

6 - Após a instalação de todos os softwares sugere-se abrir e compilar um arquivo .tex para testar a instalação

7 - Software simples para converter imagens jpeg para eps: Jpeg2Ps (http://gnuwin32.sourceforge.net/packages/jpeg2ps.htm)
